## Reporting a Vulnerability

If you discover a security vulnerability, please do not report it publicly. 

- **Contact:** Please email [mdaminjilani313@gmail.com](mailto:mdaminjilani313@gmail.com).
- **Process:** You can expect an initial response within 48 hours. We will provide updates throughout the process of fixing the issue.
- **Privacy:** We ask that you keep the vulnerability private until it is officially resolved.
